## 목록

* 순서가 없는 목록 ( `*` 누르고 입력 )
* 순서가 없습니다.
  * tab해서 들어가시고
  * 엔터해서 계속 쓰시다가
* shift+tab해서 나오시고
* 엔터+엔터하면 나가짐

이렇게

1. 순서가 있는 목록
2. 얘도 tab해보면
   1. 들어가지고
   2. 엔터해서 계속쓰다가
3.  shift+tab 해서 나오시고

## 코드 블록

인라인 코드 블록 `def` 

```python
# 주석
print('hello');
def foo():
    return True
```

여기 사이 글 쓰려면 </> 눌러서 엔터로 공간확보하면됨  ( ctrl+enter도 됨 )

```html
<!-- html(주석) -->
<h1>
    hello
</h1>

```
